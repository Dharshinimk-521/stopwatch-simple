# Stopwatch Web App

A simple and interactive **Stopwatch application** built using **HTML, CSS, and JavaScript**.
It allows users to measure time and record intervals (tasks) with local storage support.

---

## Features

- Start, Stop, and Reset stopwatch
- Displays time in **HH:MM:SS format**
- Add time intervals as tasks
- Delete last recorded interval
- Saves intervals using **localStorage** (persists after refresh)

---

## Tech Stack

- JavaScript (DOM, Events, LocalStorage)

## How It Works

- `setInterval()` is used to update time every second
- `clearInterval()` stops the timer
- Intervals are stored in an array and saved using:

  ```js
  localStorage.setItem("intervals", JSON.stringify(intervals));
  ```

- Data is retrieved using:

  ```js
  JSON.parse(localStorage.getItem("intervals"));
  ```
